# SuperRent
CPSC 304 group project: Embedded SQL, JDBC, Java and JFrame (GUI) 

* Required project for CPSC 304: Introduction to database design. No prior code was provided except for a few resources.

-- Dependencies --

* LGoodDatePicker JAR file for showing date and time in GUI 

* JDK 11.0.4

-- Assumptions -- 

* The person can not rent more than one car in an interval
